Literals vs. References

Literals are the instances of data structures, while references point at them.

Primitive variables are passed by value (literal), while objects are passed by reference.

This prevents unnecessarily large copies of objects from being made.

Notable findings:

  • The number 0 is treated as "false".
  • Empty string is also "false"
  • Empty arrays and objects are treated as "true"

Takeaways:

  • Never test the existence of a number or string without type checking (typeof).
  • References are treated as true
