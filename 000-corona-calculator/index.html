The Objective:

  • Overview of writing a simple webapp using vanilla javascript
  • Use JS in the console to gather some data
  • Utilize 3rd party libraries where necessary

The Project:

Create a calculator that takes in the:
  • Growth Rate: how many days before the number of cases doubles
  • Starting Point: how many cases to start the calculation at
  • Duration: number of days to project until

and generate a line chart of the projected growth.

Data should be retrieved from Our World in Data here. by running the script in data.js and copying the return value.
